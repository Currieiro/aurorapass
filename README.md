# AuroraPass

AuroraPass es un gestor de contraseñas de ejemplo:

- Bóveda cifrada localmente con una **contraseña maestra**.
- App iOS en **SwiftUI**.
- Inicio de sesión con **Usuario + Contraseña maestra** en iOS.
- Extensión para **Google Chrome** (Windows y Mac) que puede leer la misma bóveda cifrada.

> Proyecto educativo, no auditado para uso en producción. Úsalo bajo tu responsabilidad.
